## 📋 What is filament-translatable?

This tool adds extra features to the Filament plugin you might already use with Laravel apps. It helps you manage translations better. You will see flags for languages, clear markers showing translation progress, and a cleaner user experience for working with multiple languages. 

You don’t need to know coding or programming. This guide will walk you through how to get it onto your Windows computer and start using it.

## 🧩 How to Use filament-translatable with Laravel and Filament

Even though this guide is for beginners, filament-translatable works alongside Laravel and Filament. Here is a simple overview of what to do:

1. **Open a Command Window**  
   Press the Windows key, type **cmd**, and press Enter.

2. **Navigate to Your Project Folder**  
   Use the `cd` command followed by the path to your Laravel project. For example:  
   ```
   cd C:\Users\YourName\Documents\MyLaravelApp
   ```

3. **Install filament-translatable**  
   Your Laravel project uses Composer, a tool for managing PHP packages. To add filament-translatable, you would type:  
   ```
   composer require allam2002/filament-translatable
   ```

4. **Publish the Plugin Configurations**  
   Run this command to create needed files in your project:  
   ```
   php artisan vendor:publish --tag=filament-translatable-config
   ```

5. **Add the Plugin to Filament**  
   Open your Laravel app’s Filament service provider file and add filament-translatable to your list of plugins.

6. **View Translation Status in Your App**  
   Launch the Filament admin UI and you should see the new translation status indicators and language flags.

## 🛠 Troubleshooting Tips

- If the `composer` command is not recognized, make sure Composer is installed on your system and added to your system PATH. You can download it from [getcomposer.org](https://getcomposer.org).
- If you get permission errors when running commands, try running the Command Prompt as Administrator (right-click > Run as administrator).
- Make sure you use the correct path when navigating with the `cd` command in the command window.
- Check your internet connection if downloads stop or fail.
